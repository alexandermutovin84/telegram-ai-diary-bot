Импорт OpenAI (Whisper) в AmneziaVPN — macOS
============================================

Файл рядом: amnezia-openai-chatgpt.ip-list.json
Список IPv4 (CIDR) для портала chatgpt.com с https://iplist.opencck.org/ в формате импорта Amnezia (список обновляют командой: npm run amnezia:iplist).

Почему ChatGPT в браузере работает, а бот ругался на OpenAI
-------------------------------------------------------------
Сайт в браузере (chatgpt.com) и программный API для голоса (api.openai.com) — разные адреса и иногда разные маршруты.
У многих VPN IPv6 не попадает в туннель: браузер может идти по IPv4, а Node.js — по IPv6 и «терять» связь с API.
В коде бота включён приоритет IPv4 для исходящих запросов; после обновления перезапусти бота (Ctrl+C и снова npm run dev).

Шаги в AmneziaVPN
-----------------
1. Подключись к VPN как обычно.
2. Настройки → Split tunneling → сайты (IP), не приложения (на macOS только по IP).
3. Режим: «Только сайты из списка — через VPN» (или «Добавить импортированные к существующим», если уже есть список для Telegram и т.д.).
4. Меню «⋮» → импорт → выбери amnezia-openai-chatgpt.ip-list.json.
5. Включи split tunneling для сайтов и переподключи VPN.

Проверка на компьютере с ботом
------------------------------
  npm run check-openai

Если после импорта в терминале всё равно: 403 «Country not supported»
---------------------------------------------------------------
Запрос до OpenAI уже доходит, но OpenAI запрещает API для страны твоего **выходного** IP (часто так при выходе в РФ).

Список IP из этого файла тут не поможет — важна **страна сервера** в Amnezia.

Сделай так:
  1) В Amnezia отключи VPN.
  2) Выбери другой сервер / локацию с выходом в **США, ЕС, Японию** и т.п. (не РФ), снова подключись.
  3) На время проверки можно отключить split tunnel «только выбранные сайты» — чтобы весь трафик шёл через эту зарубежную ноду.
  4) Снова: npm run check-openai — должно быть «OK».

Если в Amnezia доступен только российский выход — для Whisper/OpenAI нужен **свой VPS за границей** в Amnezia или другой VPN с зарубежным выходом.
